# Kiroku

Kiroku is an offline-first Japanese study app for Hiragana, Katakana, and imported Anki decks.

It includes kana drills, spaced repetition review, custom deck import, and browser-to-browser sync through the minimal Go API.

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run lint
npm run build
```

## Backend

```bash
cd backend
go test ./...
```
